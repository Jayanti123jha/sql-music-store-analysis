# sql-music-store-analysis
SQL-based data analysis project on a music store database using complex queries, joins, CTEs, and window functions.

## Project Overview

This project analyzes pizza sales data to derive meaningful business insights related to revenue generation, customer ordering behavior, and product performance.

## Objectives

* Retrieve total orders and total revenue.
* Identify best-selling and highest-priced pizzas.
* Analyze order distribution by time and date.
* Determine category-wise and size-wise sales performance.
* Analyze cumulative revenue trends.
* Identify top-performing pizzas based on revenue.

## SQL Concepts Used

* Joins
* Aggregate Functions
* GROUP BY and ORDER BY
* Subqueries
* Common Table Expressions (CTEs)
* Window Functions
* Ranking Functions

## Tools Used

* PostgreSQL
* pgAdmin
* SQL

## Key Insights

* Peak ordering hours and busiest days.
* Revenue contribution by pizza category.
* Most ordered pizza types and sizes.
* Top-performing pizzas based on revenue.
* Cumulative revenue trends over time.

## Dataset

The dataset contains information about orders, order details, pizzas, and pizza categories.
